# Business Loan Calculator

A comprehensive business loan calculator that helps small business owners compare different loan types and calculate monthly payments.

## Features

- Calculate monthly payments for various loan types
- Compare term loans, lines of credit, SBA loans, equipment financing, and invoice factoring
- Real-time calculations with amortization formulas
- Educational content about business loans
- Mobile-responsive design
- SEO optimized

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Static site generation

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The static site will be generated in the `out/` directory.

## Deployment

This site is deployed to Vercel and accessible at: https://business-loan-calculator.larrys-world.com
